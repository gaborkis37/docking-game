# Asset Tracking

This file tracks all non-placeholder assets used by the project.

## Asset Policy

Agents must either:

1. Obtain assets from approved public or openly licensed sources and document them here.
2. Stop and provide exact instructions for obtaining missing assets.

Never invent asset sources.

## Preferred Sources

- NASA public-domain imagery.
- ESA media resources.
- Public-domain textures.
- Openly licensed low-poly spacecraft models.

## Development Priority

Gameplay comes first. Use placeholder geometry and simple procedural visuals until gameplay is validated.
